Aneya Abacus Revision Portal

A secure, proctored assessment platform developed for Aneya Abacus & Vedic Math Centre, Katraj, Pune. This application enables students to perform daily mental math revisions with automated grading, real-time proctoring, and comprehensive progress tracking.

#Key Features

🔐 Security & Proctoring
Anti-Cheat Enforcement: Prevents tab switching and unauthorized browser activity.
Secure Environment: Built-in safeguards against Developer Tools (F12), right-click, and copy-paste attempts.
Camera Proctoring: Mandatory camera access with a real-time "Pixel Variance" algorithm that detects if the lens is covered,                       blocked, or obstructed by any object.

📊 Progress Tracking
Parent Dashboard: A dedicated portal for parents to view student performance metrics.
Progress Visualization: Integrated data visualization providing daily score trends via interactive line graphs.
Automated Reporting: Secure, automated email reporting sent to parents immediately upon exam submission.

⚡ Technology Stack
Frontend: HTML5, CSS3, JavaScript (Vanilla ES6+).
Backend Engine: Google Apps Script (Serverless API).
Database: Google Sheets (NoSQL implementation).
Analytics: Chart.js integration for data representation.

Deployment Information
This application is hosted via GitHub Pages. All data processing is managed via the Google Workspace environment, ensuring high reliability for student assessment workflows.

Built for Aneya Abacus & Vedic Math Centre, Katraj, Pune.
